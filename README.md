## Why JSON Helper?

Working with deeply nested JSON can be overwhelming. Whether you're debugging an API response, reviewing a configuration file or exploring a dataset, reading raw text is slow and error-prone.

JSON Helper gives you a **real-time visual representation** of your data so you can focus on what matters — the structure and the values — without getting lost in brackets.

## What it does

- **Interactive tree visualization** — Your JSON is rendered as a navigable node graph with clear parent-child relationships.
- **Live editing** — Modify values, add or remove keys directly from the visual graph or the built-in code editor. Changes sync instantly.
- **Split, editor & visualizer modes** — Work the way you prefer: side-by-side, code only or graph only.
- **Search & filter** — Quickly locate any key or value across the entire structure.
- **Undo / Redo** — Experiment freely. Every change is tracked and fully reversible.
- **Zoom & pan** — Navigate large structures effortlessly with smooth zoom and drag controls.
- **Horizontal & vertical layouts** — Switch tree orientation to best fit your data shape.
- **Persistent session** — Your work is automatically saved locally so you never lose progress.

## Who is it for?

- **Developers** debugging API responses or config files.
- **QA engineers** inspecting test payloads.
- **Data analysts** exploring structured datasets.
- **Anyone** who works with JSON and wants a clearer view.

## Getting started

Paste or type your JSON into the editor — the visualization updates in real time. That's it.
